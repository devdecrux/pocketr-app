# Roadmap

Pocketr is built around a simple idea: make personal and household budgeting clear, trustworthy, and easy to run yourself. This Roadmap gives an example structure for the areas planned for future work.

The Roadmap will change as the project evolves. Items listed here are direction-setting, not fixed release promises.
